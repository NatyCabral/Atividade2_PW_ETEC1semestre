Atividade Prática de HTML




Instruções





Tema livre lembrado a proposta do tópico primeiro. Enviar a pasta de toda criação do site compactada e para detalhar qualquer explicação criar um arquivo informativo de texto do tipo README.txt.

  • Criar um site que tenho uma proposta real. Exemplo site para uma escola

  • O site precisa ter no mínimo 5 páginas (htmls)

  • O site precisa conter as seguintes TAG:OL,UL, IMG,A,DIV,P,SPAN,H1,/H1

  • Todo o design do site precisa estar em um arquivo CSS separado das páginas. Utilizar a TAG LINK ...

  • Estruturar o site utilizando as pastas e a chamada principal com o arquivo INDEX.HTML

  • Novos comandos durante as aulas podem ser colocados no site

  • Podem estabelecer grupos de até dois participantes e os dois participantes devem enviar a atividade





O prazo para envio da atividade é dia 04/04/22, pelo Teams.
